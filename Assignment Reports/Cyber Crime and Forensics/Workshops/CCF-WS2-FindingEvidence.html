Workshop 2 - Finding evidence «Forensics analysis»

Given:
Virtual machine by this: http://teacher.os3.site/CCF/Exercise_2_3/ where there is a disk image.

All the files were downloaded from the link above and the T2-Forensics-VM.v3.1.vbox file was opened in the Virtual Box program.

For best performance, the virtual machine configuration file has been changed and the following values have been set:

  • RAM: 4 Gb
  • CPU: 2

Work assignment:

  1. Parse the image to list all files and create the timeline from the prepared listing.

     $ fls -m "C:/" -r -p -l /data/Evidence/primary-disk.dd > /data/Evidence/fls-body.txt


     $ mactime -d -b /data/Evidence/fls-body.txt 2010-01-01 > /data/Evidence/timeline.csv
