wave file sent by HackRF and listening with RTL dongle

just the noise on stdout @433Mhz,

rtl_sdr -f 433000000 -s 44100 -

sending the WAVE file with its pre-defined rate (44100 samples per second) into the air with HackRF @433Mhz,

hackrf_transfer -t /var/tmp/sample.wav -f 433000000 -f 44100

the noise mixed with the wave file,

rtl_sdr -f 433000000 -s 44100 -

2.3. Can you retrieve the library added logic functionality by sniffing the stream?
